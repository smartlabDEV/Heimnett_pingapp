Hent data med fetch()

Hent liste over overvakede tjenester og bygg din egen visning.

javascript 
// Hent alle tjenester
const res = await fetch('http://DIN-SERVER:8000/api/targets')
const { targets } = await res.json()

// Filtrer pa kategori
const gaming = await fetch('http://DIN-SERVER:8000/api/targets?category=Gaming_Steam')
const { targets: gamingTargets } = await gaming.json()

// Soek
const netflix = await fetch('http://DIN-SERVER:8000/api/targets?search=netflix')

Hent data med requests

python 
import requests

BASE = 'http://DIN-SERVER:8000'

# Alle tjenester
r = requests.get(f'{BASE}/api/targets')
targets = r.json()['targets']

# Filtrer pa kategori
gaming = requests.get(f'{BASE}/api/targets', params={'category': 'Gaming_Steam'})

# Alle kategorier
categories = requests.get(f'{BASE}/api/categories').json()

Hent latency-data fra Grafana

For full kontroll — hent raa latency-data og tegn dine egne grafer.

javascript 
// Generer API-nokkel i Grafana: Configuration > API keys
const GRAFANA = 'http://DIN-SERVER:3000'
const API_KEY = 'din-api-nokkel'

// Hent dashboard
const res = await fetch(`${GRAFANA}/api/dashboards/home`, {
  headers: { 'Authorization': `Bearer ${API_KEY}` }
})

// Hent alle datasources
const ds = await fetch(`${GRAFANA}/api/datasources`, {
  headers: { 'Authorization': `Bearer ${API_KEY}` }
})

API Referanse

Klikk på en rad for å kopiere URL-en.

Metode Endepunkt Beskrivelse
GET /api/targets Alle tjenester (150+)
GET /api/targets?category=X Filtrer på kategori
GET /api/targets?search=X Søk i navn og host
GET /api/categories Alle kategorier
GET /api/categories/{id} En kategori med targets
GET /api/hosts Enkel host-liste for dropdown
GET /api/health Helsestatus
GET /docs Swagger UI (interaktiv)
